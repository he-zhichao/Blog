# 前端技术说明

采用 `Vue2.0` 的 **SSR** 框架 —— **Nuxt.js**，来作为整个项目的前端主体框架。

并按照开发需求，引入 **ElementUI**、**Vuesax**、**VantUI** 等前端组件库；引入 **ColorUI** 前端CSS库；引入 **Axios**等。

同时按照项目具体的需求，引入了之前个人开发项目 [**Concise**](https://github.com/Jing-Wenxing/concise) 中的部分技术验证与实现。
